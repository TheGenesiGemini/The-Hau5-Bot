# The-Hau5-Bot

## Descripción

Bot avanzado para automatización de trading en Binance y gestión de códigos de Telegram. Repositorio seguro y profesionalizado por Copilot.

## Seguridad

**¡No subas tu archivo `.env`!**  
Todos los datos sensibles van en `.env` (usa el ejemplo `.env.example`).

## Instalación

1. Clona el repo
2. Copia `.env.example` a `.env` y agrega tus datos
3. Instala dependencias:  
   ```bash
   pip install -r requirements.txt
   ```
4. Ejecuta el bot

## Configuración

- Todo se configura en `.env` (API keys, tokens, usuario, etc.)
- Los chats están definidos en `core/config.py` y puedes editarlos.

## Compilación a EXE

Usa los scripts en la carpeta `BUILD/` según si quieres consola o no.
